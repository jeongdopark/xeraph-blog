## 3. Layout

Layout은 요소의 기하학적 속성을 찾는 과정입니다.

DOM과 ComputedStyle 결합하여 Layout Tree를 생성합니다.

레이아웃 트리에서 각 노드는 위치와 크기에 대한 정보를 가지고 있으며 웹 페이지에 보이는 요소에 대한 것만을 포함합니다.

display : none과 같이 화면에 표시되지 않는 속성은 포함되지 않습니다.
